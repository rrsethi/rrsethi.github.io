Ubuntu Version Upgrade Guide

This guide will walk you through the steps to upgrade your Ubuntu system to the latest LTS version.

Steps to Upgrade

1. Update the package list and upgrade the installed packages:

sudo apt-get update -y && sudo apt-get upgrade -y
sudo apt update
sudo apt upgrade -y

2. Perform a distribution upgrade:

sudo apt dist-upgrade -y

3. Install the release upgrader core:

sudo apt-get install ubuntu-release-upgrader-core -y

4. Modify the release upgrades file:

sudo nano /etc/update-manager/release-upgrades

Change Prompt to lts.

5. Run the release upgrade:

sudo do-release-upgrade

6. Final update and upgrade after the release:

sudo apt-get update -y && sudo apt-get upgrade -y

Follow these steps to ensure a smooth upgrade of your Ubuntu system to the latest LTS version.
